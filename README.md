# DataStructuresImplemenationForBeginners

This repo helps teaches how to implement some of the commonly used datastructures in plan Java.It contains:

1. LinkedList
2. Stack
3. Queue
4. Tree (with n nodes)
5. Binary Tree
6. Binary Search Tree
      i.  Level Order Traversal to print tree as 1 ! 2,3 ! 4,5 !
      ii. InOrder, PreOrder and PostOrder Traversal
7. Linear Search
8. Binary Search
9. Queue Via stacks
10. Map
11. Set Using Map 
12. BFS traversal
13. DFS traversal
14. Bubble Sort
15. Insertion Sort
16. Selection Sort
17. Radix Sort
18. Bucket Sort
19. Merge Sort
20. Quick Sort
21. Heap Sort
22. LRU Cache
